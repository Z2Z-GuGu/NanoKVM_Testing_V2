# NanoKVM Pro 产测程序 V2

NanoKVM Pro 产测 V2 是一个基于 Tauri 框架开发的桌面应用程序，专为 NanoKVM Pro 设备的生产测试而设计。该项目使用 React 19、TypeScript 和 Vite 构建现代前端界面，结合 Tauri 提供的系统级 API，实现对硬件设备的底层控制与测试，通过现代化的 UI 设计和丰富的组件库（如 Radix UI、Recharts），为用户提供直观易用的操作界面，确保产品质量和生产效率。

NanoKVM-Pro Testing V2集成了串口通信、GPIO 控制、网络配置等功能模块，为生产环境提供高效、稳定的自动化测试解决方案。

NanoKVM-Pro Testing V2使用Tauri开发，其中大部分组件可以支持跨平台使用，但少量组件如ipconfig/Windows api等仅可以在Windows下编译和使用，如果要考虑平台迁移，需要解决这些平台相关的依赖

该产测软件具有以下特点：
1. 产测程序通过高度并行测试大幅提高产测效率；
2. 通过UI直观展示产测进度；
3. 为了避免产测人员的等待时间不确定性，将所有产测项目应该测试的时间/次数写入到产测程序中，超时即可触发项目的测试失败；
4. 考虑到产测人员的工作枯燥问题，软件加入弹窗机制，如果遇到不确定的项目需要手工点击弹窗；
5. 标签打印机制：同时解决了产测时写标签费时问题和发货版本问题，同时标签内容更加详尽，方便后续维护
