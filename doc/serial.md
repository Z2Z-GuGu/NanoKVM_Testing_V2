# 在src-tauri/src/threads/serial.rs中实现串口管理线程

文件架构：

+ 定义PID/VID波特率
+ 定义收发两个队列
+ 连接指定（PID/VID）串口函数
+ 串口发送函数（向写队列写）
+ 串口接收函数（从读队列读）
+ 串口管理线程函数
  + 如果没有连接就连接指定（PID/VID）串口
  + 如果连接了就交替读写串口到队列（读不到数据就不往队列里写（也就是所谓超时，是正常的，不要退出）；如果写队列没数据就不往串口写）
  + 如果读的时候报错（连接断开）就断开连接，并回到第一步，循环
+ 串口任务线程函数
  + 启动串口管理线程
  + 循环执行以下任务：
    + 串口发送“test”
    + 串口读取
    + sleep 1
